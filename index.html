Project Summary

Scott M. Downey

Knock Knock Server

Multi-threaded simulation of a server that clients can connect to to receive knock-knock jokes. The server randomly selects a joke in a pre-defined library and the client can interact with the server, replying to the jokes as necessary. Written in Java utilizing the Swing GUI.

Vending Machine Simulation

Object-oriented implementation of a vending machine simulation in C++. Simulates various operations, such as entering coins, checking prices, retrieving the product purchased, etc.

Hangman

Simple game of Hangman written in Java, utilizing the Swing GUI. A random word is selected from a pre-defined word library and the user attempts to guess each letter of the word before the full hangman picture is drawn.

Airplane Monitoring Simulation

Implements a simulation of an airplane monitoring system. Various types of signals are implemented, as are sensors to read their meaning.

Doubly Linked List

Implementation of a doubly linked list data structure in C++. Programmed in a generic-style using templates and includes an associated 'iterator' class to further help compatability with the C++ Standard Template Library.

Big Int Library

Library that allows the use of integer-type numbers larger than allowed for the built-in type 'int', 'long' and 'long long'. Supports operations of addition and subtraction. Also allows for comparison operations and insertion (>>) and extraction (<<).
